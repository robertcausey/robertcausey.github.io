  2. Protocol for this run
    1. Coat plates
      1. Concentration of MAB212P: 2μg/mL
      2. Amount added to each well: 50μL
      3. Time: overnight
      4. Temp: 4C
      5. Coating Buffer: PBS 0.15M, pH 7.6 (1 X PBS)

    2. Wash plates
      To remove non adsorbed MAB212P
      1. Wash buffer:
        1. Buffer type: 1 X PBS with 0.05 % tween
        2. Concentration: 0.15M
        3. pH: 7.6
        4. Tween concentration: 0.05 %

      2. Number of washes: Once
      3. Fill: 300 μL

    3. Block plates
      1. Blocking agent
        1. Buffer type: PBS
        2. Concentration: 0.15M
        3. pH: 7.6
        4. Skim milk concentration: 1 %

      2. Duration of blocking
      3. Temp of blocking
    4. Wash plates
      To remove excess blocking agent
      1. Wash buffer:
        1. Buffer type: PBS with Tween 20
        2. Concentration: 0.15M
        3. pH: 7.6
        4. Tween concentration: 0.05 %

      2. Number of washes: once

    5. Store plates
      1. Temp
      2. wet or dry
      3. Duration

    6. Addition of sample
      1. Source of sample
      2. Date collected

        3. pH

      5. Duration of incubation
      6. Temp of incubation

        1. Buffer type
        2. Concentration
        3. pH
